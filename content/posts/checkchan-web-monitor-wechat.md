前阵子发现一个很实用的开源工具，能监控网页内容变化并推送通知到微信。比如你想监控某个商品价格变动、抢购页面更新、或者某个 API 返回值变化，装上浏览器插件就行。配合自架云端，电脑关机后监测任务照样跑。

项目地址 `https://github.com/easychen/checkchan-dist`

---

## Check酱是什么

Check酱是方糖气球出品的网页内容监控工具，包含一个 Chrome/Edge 浏览器插件和可以自行架设的云端服务。它的核心能力是：**监测网页内容变化，异动即时推送到微信**。

和普通的网页监控不同，Check酱通过浏览器插件运行，完全模拟用户的真实浏览行为。这意味着它可以监控需要登录的页面、动态渲染的 JavaScript 内容、以及各种复杂的网页结构——这些是 curl 和爬虫搞不定的。

## 核心功能

### 可视化选择器监控

不需要写 XPath 或 CSS 选择器，直接在页面上**点击选择**要监控的内容区域。支持监控页面上的任意文本内容。

### 微信推送通知

配合 [Server酱](https://sct.ftqq.com)，监控到变化后即时推送通知到微信。也可以通过其他渠道推送。

### 自架云端

浏览器关掉后监测任务就不跑了？用 Docker 自架云端就能解决。监测任务同步到服务器后，定时运行，不受本地开关机影响。

### 完全模拟用户行为

因为是浏览器插件后台打开网页，可以自动延续登录态。那些需要登录才能看到的页面，Check酱也能正常监控。

## 快速上手

### 方式一：浏览器插件独立使用

下载插件包后，打开 Edge/Chrome 的插件页面，开启「开发者模式」，点击「Load Unpacked」选择解压目录即可。

> ⚠️ 插件可以独立使用，但关掉浏览器后定时监测任务不会执行。需要持续监控请配合云端。

### 方式二：自架云端（推荐）

一行 Docker 命令部署云端服务：

```bash
docker run -e API_KEY=你的安全密码 \
  -e TZ=Asia/Shanghai \
  -e ERROR_IMAGE=NORMAL \
  -p 8088:80 \
  -v $PWD:/data \
  -d ccr.ccs.tencentyun.com/ftqq/checkchan:latest
```

部署完成后在插件设置中填入云端地址，插件会自动将监测任务同步到服务器。

## 实用场景

- **抢购/秒杀监控**：监控电商页面库存或价格变化，第一时间收到通知
- **文章发布监控**：监控博客或论坛是否有新文章发布
- **API 健康检查**：监控后端 API 的返回值或 HTTP 状态码
- **RSS 更新提醒**：监控订阅源是否有新内容
- **竞品动态监控**：追踪竞品网站的内容变化
- **价格监控**：商品价格变动即时提醒
- **招聘/政策监控**：网页内容更新第一时间知晓

## 架构设计

Check酱采用「插件 + 云端」的分层架构：

1. **浏览器插件**：负责页面渲染和内容对比。利用浏览器环境自动处理 JavaScript 渲染、Cookie/登录态延续
1. **自架云端**（可选）：Docker 部署，接收插件同步的监测任务，定时执行。电脑关机后监测不中断
1. **Server酱**：消息推送通道，将变化通知发送到微信
> 💡 插件是核心，云端是增强。如果你不需要 24 小时监控，只用插件就够了。

---

## 部署注意事项

- 云端镜像为 **x86 架构**，ARM 架构需自行拉取源码构建
- 挂载的 `/data` 目录需要**写权限**
- `API_KEY` 设置一个安全密码，不要带 `$` 符号
- `ERROR_IMAGE=NORMAL` 表示出错时使用普通通知而非图片
- 如果用腾讯云镜像拉取慢，可以换 Docker Hub 源
