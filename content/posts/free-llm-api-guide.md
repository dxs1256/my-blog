## 🇨🇳 一、国产大模型：免费额度最猛

### 1. 通义千问 Qwen（阿里云）⭐⭐⭐⭐⭐

每月1000万输入Token + 100万输出Token（永久免费）

- 模型：Qwen-Turbo（速度快、质量不错）
- 申请：注册阿里云 → 灵积模型服务 → 获取API Key
- 适合：日常对话、文本处理、翻译、摘要
> 💡 1000万Token≈处理750万字，普通人一个月根本用不完

### 2. 智谱 GLM（清华系）⭐⭐⭐⭐⭐

每月500万输入Token + 100万输出Token（永久免费）

- 模型：GLM-3-Turbo / GLM-4-Flash
- 申请：open.bigmodel.cn → 注册 → 获取Key
- 适合：中文写作、知识问答、代码生成
### 3. DeepSeek（深度求索）⭐⭐⭐⭐⭐

注册送500万Token + 超低价 ¥1/百万Token

- 模型：DeepSeek-V3.2、DeepSeek-R1（推理模型）
- 申请：platform.deepseek.com → 注册
- 适合：编程、数学推理、复杂分析
> 🔥 R1推理能力部分场景超过GPT-4o，价格只有1/50

### 4. Kimi（月之暗面）⭐⭐⭐⭐

128K超长上下文 — 一次读一本书

- 模型：Moonshot-v1-128k

- 适合：长文档分析、论文阅读、全书摘要

## 💡 三、白嫖攻略：怎么最大化免费额度？

▲ 按场景分配不同模型，免费额度最大化

### 🎯 策略1：按场景组合使用

- 日常对话 → 通义千问（免费额度最大）
- 写代码 → DeepSeek V3.2（编程能力最强）
- 长文档 → Kimi 128K（超长上下文）
- 多模态 → Gemini Flash（免费+图片理解）
- 复杂推理 → DeepSeek-R1（推理能力最强）
- 英文任务 → Claude Haiku（英文表达最佳）
> 🔥 组合下来，每月免费额度超过3000万Token

### 🎯 策略2：用API不用网页版

API的免费额度和网页版是独立的。注册开发者账号拿API Key，额度通常更多。

### 🎯 策略3：蹲新模型上线

每个新模型上线时通常给大额推广免费额度，第一时间薅新用户福利。

## 🚀 四、5分钟上手API调用

Step 1：注册平台账号（阿里云/智谱/DeepSeek）

Step 2：在控制台获取API Key

Step 3：发送HTTP请求（Python/curl/在线工具都行）

大部分平台都兼容OpenAI格式 — 换个地址和Key就行，代码几乎不用改。
