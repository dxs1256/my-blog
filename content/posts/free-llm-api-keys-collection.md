现在各家大模型的 API 价格虽然一直在降，但想用上 GPT-5.4、Claude 这些顶级模型，门槛还是在那——绑定信用卡、充值、按量付费，国内学生党和刚入门的开发者经常卡在第一步。

## 💡 怎么个免费法

这个项目的思路很简单直接：作者把自己账号里用不完的 LLM API 额度拿出来公开分享，生成可用的 API Key 放在 GitHub 仓库里，每时每刻更新。你直接复制就能用。

覆盖的模型挺齐全：

- GPT-5.4 / GPT-4o
- Claude 系列
- Gemini 系列
- Grok
- 还有一些国内主流模型
每个 Key 的有效期和剩余额度都有标注，先用先得。

![](https://i.ibb.co/wN6nCd2Q/1d19a6190e10.jpg)

## 🎯 适合什么场景

- **学生党体验** — 没有国际信用卡，想试试 GPT-5.4 效果怎么样，直接拿走就用
- **开发测试** — 写了个小工具要调 API，没必要为了测试专门开个付费账号
- **模型对比** — 同一个 Prompt 在不同模型上的输出对比，省去轮流充值的麻烦
## ⚠️ 注意

因为是公开分享的 Key，有几个点需要注意：

- Key 随时可能被其他人用光额度，不适合用在正式产品里
- 不要用它处理敏感数据，毕竟是共享 Key
- 建议复制后尽快使用，先到先得
