最近在做安全排查时，需要确认几个账号的关联信息，手工一个个平台去搜效率太低。结果发现了一个叫 **Maigret** 的开源工具，只要输入一个用户名，3秒内就能自动检索 3000 多个网站，把目标人物在互联网上留下的「赛博马甲」全部扒出来，整理成一份详细的数字档案。

这个项目在 GitHub 上已经拿到 2.8 万 Star，五一期间日均增长超过 1000 颗，连续多日霸榜。今天把这个工具分享出来。

## 项目信息

Maigret 项目地址：

```plain text
https://github.com/soxoj/maigret
```

项目名称灵感来源于法国侦探小说中的神探儒勒·梅格雷——他的探案方法不是炫技，而是「理解不同人物的性格及其相互作用」。Maigret 同样做到了这一点：不只是搜索，而是「顺藤摸瓜」式的情报挖掘。

---

## 核心能力

- **用户名搜索**：只需一个用户名，自动检索 3000 多个网站
- **递归挖掘**：发现关联用户名后自动开启递归检索，顺藤摸瓜挖出更庞大的隐藏身份网络
- **信息提取**：从页面中提取真实姓名、注册时间、关联第三方链接等信息
- **标签过滤**：内置标签系统，可按类别（如 social）或区域锁定搜索范围
- **多种导出格式**：支持 HTML、JSON、PDF 报告，结果一目了然
- **AI 分析模式**：支持 `--ai` 参数，将原始数据自动转化为高熵情报摘要
- **零依赖**：全程无需任何 API 密钥，完全开源免费
---

## 使用方法

### 安装

```bash
pip install maigret
```

### 基本用法

最简单的用法，只需一个命令：

```bash
maigret YOUR_USERNAME
```

它会自动在 3000 多个网站上搜索这个用户名，把所有匹配的结果整理成报告。

### AI 分析模式

加上 `--ai` 参数，Maigret 会自动分析原始数据，生成高熵情报摘要：

```bash
maigret YOUR_USERNAME --ai
```

### 指定报告格式

可以指定输出格式为 HTML、JSON 或 PDF：

```bash
maigret YOUR_USERNAME --html
maigret YOUR_USERNAME --json
maigret YOUR_USERNAME --pdf
```

## 工作原理

Maigret 的核心逻辑是：给定一个用户名，它会遍历内置的 3000 多个网站数据库，逐个检测该用户名是否存在。如果在某个网站上发现了匹配结果，它会进一步提取页面中的关键信息（如真实姓名、注册时间等）。

更强大的是，如果提取过程中发现了关联的「另一个用户名」，它会自动开启递归检索，继续在 3000 多个网站上搜索新的用户名，形成一个不断扩展的身份网络图谱。

## 注意事项

- **合法使用**：仅限合法的安全研究和授权排查，未经授权的个人信息收集可能违法

- **网络环境**：由于需要访问大量网站，建议在稳定的网络环境下使用
