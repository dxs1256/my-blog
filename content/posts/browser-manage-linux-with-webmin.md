每次要改个防火墙规则、加个定时任务、配个虚拟主机，第一反应就是 SSH 连上去敲命令。习惯了倒也不觉得麻烦，但次数多了确实有点累，尤其是改配置文件时一个手误可能把服务搞挂。

**GitHub：**[https://github.com/webmin/webmin](https://github.com/webmin/webmin)

---

## 💡 这个面板跟宝塔不一样

Webmin 1997 年就发布了第一个版本，到现在已经 28 年了，GitHub 上 5900 Star、25000+ 次 commit，年安装量超过一百万。

它跟宝塔的思路完全不同。宝塔是把 Linux 包起来，你装它的 LNMP 一键包就得按它的路径来，卸了面板网站可能就跑不起来了。Webmin 不搞这一套——它直接读写系统原生配置文件，加一个用户改的就是 /etc/passwd，调 Apache 虚拟主机改的就是 /etc/apache2/sites-available/ 下的 conf。你把 Webmin 进程关了，服务器该怎么样还怎么样。

装完才 26MB，跑起来 100MB 左右内存，没请求时 CPU 基本不动。

## 🎯 116 个模块，覆盖你能想到的运维操作

装好之后浏览器打开 https://你的服务器IP:10000，root 登录。左边导航栏展开就是 116 个模块，几乎覆盖了日常运维的所有场景：

- **用户和组** — 增删改查，直接操作系统用户
- **防火墙** — 图形化配置 iptables/nftables
- **Web 服务器** — Apache/Nginx 虚拟主机管理
- **数据库** — MySQL/MariaDB/PostgreSQL 管理
- **定时任务** — crontab 图形化编辑
- **文件管理** — Filemin，浏览器里操作文件，内置代码编辑器
- **终端** — 浏览器里的 SSH 终端，应急用很方便
所有操作都在浏览器里点，不用记住那些冗长的命令参数。

## 🖥️ 界面现在长什么样

以前 Webmin 的界面确实劝退——灰扑扑的表格，直角边框，透着一股子世纪初 Debian 安装界面的味道。但从 2.6xx 版本开始，Authentic Theme 做了大改。

现在支持深色/浅色模式一键切换，导航栏可折叠，右上角能搜索任意模块。仪表盘的 CPU、内存、磁盘读写图换成了 Canvas 渲染，拖拽缩放流畅了不少。

## 🔧 安全性怎么样

面板直接暴露 10000 端口，安全性确实需要注意。官方也想了办法：

- 默认支持 Let's Encrypt 自动申请证书，一键开启 HTTPS
- 支持 2FA 双因素认证
- 可以绑定 IP 白名单，限制来源访问
- 建议把默认端口改掉，再加一层 Cloudflare Tunnel 代理
对于个人或小团队用，这些措施够用了。

- 帮别人管服务器，需要个直观界面的
- 想找一个不绑架环境的轻量面板，随时可卸的

## 📝 写在最后

Webmin 不是最潮的面板，但它 28 年积累下来的稳定性和原生配置管理思路，让它在一个角落里活得很好。如果你跟它理念合拍——系统是你自己的，它只是帮你操作——那它就是个趁手的工具。
