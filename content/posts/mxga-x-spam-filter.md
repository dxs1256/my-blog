刷 X 评论区时有没有这种感受——一半是色情 Bot，另一半是割韭菜广告水军，正常讨论全被压到最底下？最近发现一个叫 **MXGA** 的开源 Chrome 扩展，专治这种乱象，在后台静默帮你识别并拉黑 Spam。

🔗 ：[https://github.com/foru17/make-x-great-again](https://github.com/foru17/make-x-great-again)

---

## 🎯 一个赛博清道夫

MXGA（Make X Great Again）是一个完全开源、AI 驱动的 X 旁路 Chrome 扩展。它的工作方式很简单：你照常刷推，它在后台静默识别色情和广告垃圾，一个按钮就能调用 X 官方接口进行真·拉黑。

跟传统屏蔽工具不同，它不依赖单一规则，而是用 AI 模型+社区共建榜单双重把关。

![](https://external-content.duckduckgo.com/iu/?u=https://images.weserv.nl/?url=https://pbs.twimg.com/media/HJX7y2Ha0AA00d-.jpg?name=orig)

## ⚡ 三大亮点

**AI 后台静默识别**：不需要手动标记，扩展在后台自动分析推文内容。识别到 Spam 后一键触发 X 官方拉黑接口，干净利落。

**社区共建公榜**：维护者和社区共同维护黑白名单。3 个老 GitHub 账号 + 高置信度 AI 才能把骗子送上榜，从机制上杜绝误杀。公开榜单地址：[https://x.zuoluo.tv/](https://x.zuoluo.tv/)

**零隐私泄露**：不收集任何个人数据，不需要注册账号，源码全开。底裤护得死死的。

## 💡 未来规划

项目还在持续迭代，后续计划上线 KOL 靠谱度扫描、Profile 自动摘要、一键导出 JSON 等功能。如果你是 X 重度用户，这个赛博清道夫值得放进工具箱。
