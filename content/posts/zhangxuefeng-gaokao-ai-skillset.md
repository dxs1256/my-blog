## 📝 项目背景

中国每年超过1200万高考考生，大量来自信息资源匮乏的普通家庭。志愿填报的信息差，不应该靠花钱来弥补。

这个项目以张雪峰命名，纪念他为中国普通家庭消除教育信息差所做出的贡献。核心理念："他用一张嘴，替无数普通家庭的孩子打探了前路。我们用AI Agent，把这条通向未来的路铺向远方。"

## 📊 项目信息

- GitHub Stars：176+

## ⚡ 核心差异

不只是数据匹配工具（分数→学校），而是一套完整的决策框架：

- 帮助普通家庭用正确的思维顺序做志愿决策
- 纳入AI时代的行业前景校正
- 输出最多3个选项+明确切换条件，不给模糊备选清单
## 📚 知识库模块（8大模块）

- AI时代校正框架 - 2026-2030视角专业选择动态校正
- 专业选择决策框架 - 高确定性 vs 高风险分析
- 志愿填报操作框架 - 冲稳保策略、城市优先级
- 就业路径分析 - 五条主路径真实评估
- 院校选择参考框架 - 各分数段逻辑
- 备考策略与决策底层逻辑
- 大学在校规划 - 英语、社团、考研导师、人脉
- 新高考选科决策 - 3+1+2模式分析
## 🚀 三种使用方式

### 方式一：直接阅读

进入 knowledge/ 目录阅读 Markdown 格式知识库即可。

### 方式二：部署为 OpenClaw Skills（推荐）

接入微信/企业微信/QQ，让家长用熟悉的方式获得AI顾问服务。

### 方式三：导入 RAG 平台

支持 Dify、FastGPT、RagFlow 等平台构建问答机器人。

## 🧠 核心决策框架（四步法）

### 第一步：先定可行集

- 用分数+省份确定实际能报的院校层次
- 特殊通道（体育/艺术特长）优先核查
### 第二步：目标倒推

- 先问"最终想要什么保障"
- 再选能达到这个目标的最短路径
### 第三步：AI时代校正

- 核查每个候选专业的AI冲击评级
### 输出规范

- 最多3个选项 + 明确切换条件
- "分够X就选A，分不够就选B"
## 🔧 部署方法

```bash
# 1. 克隆仓库
git clone https://github.com/Eric-Yibo-Shen/zhangxuefeng-skillset.git
cd zhangxuefeng-skillset

# 2. 安装到 OpenClaw
openclaw skills install ./skills/gaokao-mentor.skill

# 3. 验证安装
openclaw skills list
```
