想让你的 AI Agent 看遍全网，却总是被 API Key、付费墙、403 报错卡得死死的？Agent Reach 用一条指令，就能读取、搜索 Twitter/X、Reddit、Bilibili、小红书、YouTube、GitHub 等 15+ 平台的内容，而且完全免费！

## 全平台能力覆盖

Agent Reach 支持数十个国内外主流平台，从基础的网页阅读、全网搜索，到短视频平台、社交平台、技术社区的内容获取，覆盖了绝大多数 AI 联网使用场景。

- Twitter/X — 推文内容获取与搜索
- Reddit — 帖子与评论抓取
- Bilibili — 视频内容解析
- 小红书 — 笔记内容获取
- YouTube — 字幕提取
- GitHub — 仓库与文档读取
- 微信公众号 — 文章内容抓取
- 抖音 — 视频内容获取
- 还有微博、知乎、Stack Overflow 等 15+ 平台
## 零成本核心优势

### 零 API 费用

全部基于开源 CLI 工具，通过本地 Cookie + 抓取方式访问，完全不走官方付费 API，彻底告别 API 密钥、付费墙和各种反爬限制。

### 一键安装

直接给 Agent 一条指令就能自动搞定，无需复杂配置，开箱即用。全程不用手动敲一行代码，交给 AI Agent 就能自动完成全部环境配置。

### 智能绕过限制

本地 Cookie + 抓取，稳定读取受限内容。可插拔设计，每个平台独立模块，随时更换升级。

### 安全可靠

所有数据本地运行，本地存储 Cookie，支持安全预览模式。代码开源可审查，不用担心信息泄露。

## 三步快速安装

1. 前置环境：确保 AI Agent（Cursor、Windsurf、Claude Code 等均可）拥有命令行执行权限，本地已安装 Python 3.8+
1. 一键安装：把 Agent Reach 项目地址复制给 AI Agent，输入「帮我安装 Agent Reach 工具，完成全部环境配置」
1. 验证诊断：安装完成后让 AI 执行 agent-reach doctor，自动检查所有平台连通性并给出修复建议
