想下载 YouTube 视频，以前总得面对一堆复杂的命令行参数或者满屏广告的在线工具。最近发现一个开源工具 YTSage，界面干净、操作直观，功能却一点不缩水。

项目地址：[https://github.com/oop7/YTSage](https://github.com/oop7/YTSage)

---

## 🎯 功能一览

YTSage 是一个基于 Python 的 YouTube 下载工具，核心特点就是"简单但强大"：

- **任意画质随便下**：从 144p 到 4K，选你需要的清晰度
- **纯音频一键提取**：只想听歌不想看画面，直接下载 MP3
- **整个播放列表批量拉**：一个命令拉完整个歌单或系列视频
- **多语言字幕嵌入**：下载时直接把字幕嵌进视频文件
- **SponsorBlock 自动跳广告**：识别并跳过视频中的赞助段落
- **按时段截取片段**：不需要整段视频，只下载需要的部分
- **登录 Cookie 支持**：私有视频也能正常下载

## 🔧 安装和使用

安装极其简单，一行命令搞定：

```bash
pip install ytsage
```

如果你不想用命令行，项目也提供了可直接运行的 exe 文件，下载即用。

## 💡 适用场景

YTSage 适合各种 YouTube 下载需求：

- **离线观看**：通勤路上没网的时候也能看喜欢的视频
- **音频提取**：把 YouTube 当成音乐库，下载歌曲到本地播放
- **内容备份**：自己频道的视频或喜欢的播放列表整份存档
- **剪辑素材**：下载高清视频作为后期编辑的素材来源
