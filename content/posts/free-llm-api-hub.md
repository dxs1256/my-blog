最近在搞几个小项目，需要用到不同场景的 AI 模型。每次翻 GitHub 找免费 API 都烦得不行——这个要绑信用卡，那个要手机验证，还有的文档写得跟天书似的。

后来发现了 freellm.net 这个站点，彻底治好了我的找模型困难症。

项目地址:[https://freellm.net/](https://freellm.net/)

---

---

## 🎯 它解决了什么问题

以前想找个免费的 LLM API 是这样操作的：打开 Google 搜一圈，打开三四个 GitHub 仓库看 README，对比 rate limit 和 context window，再一个个注册账号拿 API key。折腾半小时起步。

freellm.net 把这些全包了。242 个免费模型，来自 28 家厂商，全部在一个页面里列好。Google Gemini、NVIDIA NIM、OpenRouter、Groq、DeepSeek、Mistral……你能想到的基本都在。

每个模型的信息清清楚楚：上下文窗口多长、支持文本还是图文还是音频、rate limit 多少、要不要信用卡。不用挨个去翻厂商文档。

---

## 🎯 当前最值得用的免费模型

我翻了翻现在的榜单，有几个特别亮眼的：

### 📍 Gemini 3.5 Flash — 综合最强

Google 出的。100 万 token 上下文窗口，支持文本/图片/视频/音频/PDF，甚至还能推理。每分钟 15 次请求，每天 1500 次。关键是完全免费，不需要绑卡。

### 📍 NVIDIA: Nemotron 3 Ultra — 推理能力强

托管在 OpenRouter 上，100 万 token 上下文，每天 200 次免费请求。在代码和推理类任务上表现很出色。

### 📍 DeepSeek V4 Pro & V4 Flash — 性价比之选

NVIDIA NIM 上托管的，100 万 token 上下文，V4 Pro 能输出 384K token。每分钟最多 40 次请求，不限次数。唯一的小门槛是注册需要手机验证。

### 📍 Qwen3.6-27B — 多模态全能

OVHcloud 提供，131K 上下文，支持文本/图片/视频/音频/推理。匿名使用每分钟 2 次请求，够日常玩了。

## 🎯 除了搜模型，还能干什么

freellm.net 不只是一个榜单页。有几个功能我觉得做得挺到位的：

**一键配置生成**

Claude Code、Cursor、Codex、Gemini CLI——这些工具配置 API 的格式各不相同。在这个站上选好工具和模型，直接复制配置就行，不用自己手写。

**API Key 保险箱**

再也不用在文本文件里到处找 API key 了。站内提供了一个加密的 Vault，可以安全存储所有免费 API key。

**在线 Playground**

不想到处复制粘贴？直接在网页上选模型、输 API key，开聊。你的 key 不会经过他们的服务器，直接发给模型厂商。

**按使用场景筛选**

想要写代码的模型？聊天用的？能看图识图的？语音处理的？每种场景都给你标好数量——比如 Coding 场景 33 个模型，Vision 场景 46 个。

**学生/研究者**：做实验、写论文、练手，预算有限但需要多个模型轮着用。
