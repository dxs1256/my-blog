你有没有过这种经历？刚装完一个软件，明明只是本地工具，电脑风扇却突然狂转，任务管理器里多了几个叫不上名字的进程在后台偷偷联网。

项目地址 https://github.com/safing/portmaster

---

更可怕的是，它们连向哪里传输数据、传了什么，你完全不知道。

这就是咱们普通用户面对网络隐私时的真实写照——两眼一抹黑，全凭软件开发商的良心。

好在最近我在 GitHub 上挖到了一款宝藏工具 Portmaster，目前已经收获了超过 12,000+ Star。这玩意儿就像给你的电脑装了个网络摄像头，谁在联网、连向哪里、干了什么，看得一清二楚。

一、装上它，瞬间开启上帝视角

第一次打开 Portmaster，你会被它的界面惊艳到。深色的仪表盘上，左侧列出了所有正在联网的应用程序，右侧则是实时的连接详情。

功能特点：

- 开源免费的应用防火墙

- 由奥地利 Safing 团队开发

- 把复杂的防火墙配置做成傻瓜式操作

使用示例：打开 Edge 浏览器访问网页，Portmaster 立刻列出十几个连接，包括：

- 连接地区（如美国、本地局域网）

- 广告追踪域名标注

- 具体 IP 地址

- 传输协议

- 加密状态

- ASN 组织信息

二、三步上手，轻松掌控网络

第一步：下载安装

支持系统：

- Windows：支持

- Linux：支持

- macOS：开发中

安装方式：

- Windows：下载 .exe 文件，双击安装

- Linux：可用 .deb、.rpm 包或命令行安装

第二步：初始化设置

引导流程：

- 选择安全 DNS 提供商（建议 Cloudflare 或 Quad9）

- DNS 查询加密，运营商无法看到访问记录

- 默认开启广告和追踪器拦截

- 开箱即用，无需额外配置

第三步：精细化管控

操作方式：

1. 在左侧应用列表找到目标软件

2. 查看所有网络活动

3. 拨动阻止连接开关即可切断对外连接

4. 设置即时生效，无需重启软件

可设置的规则：

- 只允许连接局域网

- 禁止 P2P 连接

- 屏蔽某个国家的所有 IP

三、那些让人安心的细节

1. 系统级广告拦截

- 内置多个过滤列表

- 像 AdBlock 一样拦截广告和追踪域名

- 系统层面工作：软件内置广告、其他应用追踪请求都会被拦截

2. DNS 防泄漏

- 默认使用 DNS-over-TLS 技术

- 所有域名查询加密传输

- 进一步保护隐私

3. SPN 隐私网络（付费功能）

- 类似 Tor 的多跳加密网络

- 流量经多个节点加密传输

- 每个节点只能看到前后一跳

- 无法追踪真实身份和访问目标

四、可能的小坑

注意事项：

1. 安装时可能提示安装内核驱动或需要管理员权限，正常操作，点击允许即可

2. 个别软件可能出现联网异常，检查 Portmaster 是否误拦截，手动放行即可

3. 总体误杀率较低

五、写在最后

用了一段时间 Portmaster 后，最大的感受就是终于对自己电脑的网络活动有了主权。不再是那个软件想连就连、想传就传的被动局面，而是真正做到了我的网络我做主。

推荐理由：

- 完全开源免费

- 代码托管在 GitHub 上

- 任何人都可以审计

- 用起来心里踏实
