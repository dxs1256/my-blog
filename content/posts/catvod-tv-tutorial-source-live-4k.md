电视盒子和智能电视上用什么看片？很多人装了 App 发现点进去一片空白，4K 点开就转圈，直播也看不了——其实不是软件的问题，是没配源。

影视仓是基于 TVBox 开源框架深度优化的大屏播放器，本身不带任何资源，所有内容都靠你填的配置接口拉回来。零广告、不绑架会员、手机电视盒子都能装。

项目地址:[https://github.com/CatVodTVOfficial/TVBoxOSC](https://github.com/CatVodTVOfficial/TVBoxOSC)

🎯 **影视仓是什么**

简单说，影视仓就是一个播放器壳子——它本身不存任何电影电视剧，通过你配置的接口地址，从各个源站拉取影片数据。接口质量决定了你能看到什么、清晰度如何。

完全开源免费。凡是让你加群付费才能下载、扫码解锁高级版的，全是假包。

下载方式：官网 [http://影视仓.com/](http://xn--5mqx81b535a.com/) 或 GitHub 搜索 CatVodTV。

版本分 TV 版（电视盒子/投影仪，遥控器操作优化）和手机版（安卓端，手势控制支持）。

电视安装方法：电脑下载 TV 版 APK 拷进 U 盘，插电视 USB，打开文件管理安装即可。小米电视需开启未知来源权限，海信电视需开启商场模式权限。

🎯 **配置播放源（点播源）**

这是用影视仓的第一步，也是最关键的一步。装完打开主页空荡荡的是正常的——因为没有配置接口。

去 [https://tvbox.clbug.com/user.php](https://tvbox.clbug.com/user.php) 这个接口检测站，点全部检测，它会列出当前可用的配置接口并显示响应速度。

选一个速度快的接口，复制它的链接地址。

打开影视仓 → 设置 → 配置地址 → 粘贴链接 → 确定。

返回主页，内容就出来了。热门电影、连续剧、综艺、动漫分类清晰。

![影视仓播放源配置](https://i.ibb.co/7t6wZzCN/yingshicang-1.png)

🎯 **配置直播源**

除了点播，影视仓也支持看直播电视。配置方法和点播源类似，但需要专门的直播接口。

直播接口的获取渠道和点播源一样，在 tvbox.clbug.com 上也有专门的直播源分类。选好后在影视仓的直播配置入口粘贴即可。

配置完后在首页切换到直播 tab，就能看到央视、卫视、地方台等频道列表。

🎯 **4K 高清配置（网盘 Cookie）**

很多人卡在这一步：普通 1080P 能播，一碰 4K 就弹出「请配置网盘 Cookie」或者一直转圈。

原因是 4K 资源大多存在百度网盘里，影视仓需要借用你的网盘账号才能拉流。

配置方法：

打开影视仓，按遥控器主页键 → 选配置中心 → 找到百度网盘配置 → 设置百度网盘 Cookie。

Cookie 的获取方式：在电脑浏览器登录百度网盘网页版（pan.baidu.com），按 F12 打开开发者工具 → Network 标签 → 刷新页面 → 找到任意请求 → 复制请求头中的 Cookie 字段。

把复制的 Cookie 粘贴到影视仓的百度网盘 Cookie 配置框中，保存即可。

配置成功后，导航里的 4K 资源（木偶 4K、玩偶 4K 等）就能正常播放了。Cookie 有效期一般几天到几周，过期后重新获取一次即可。

🎯 **搭建私人直播源**

除了用现成的直播接口，也可以自己搭建直播源。开源项目 iptv-sources 可以部署在 Cloudflare Pages 上定时抓取直播源并更新订阅链接，配合影视仓使用体验更稳定。

部署方式：Fork iptv-sources 项目到 GitHub → 连接 Cloudflare Pages → 设置定时触发器（cron trigger）自动更新 → 在影视仓中填入生成的订阅链接。

自己搭建的好处是源由自己控制，不受公共接口波动影响。

🔧 **常见问题排查**

配置接口后内容不显示：换个接口试试，有些接口可能失效了。在 tvbox.clbug.com 上选最新的接口。

4K 转圈：检查百度网盘 Cookie 是否配置正确或已过期，重新获取 Cookie。

直播卡顿：切换直播源，不同源的线路稳定性不一样。也可以用 iptv-sources 自建源。

无法安装 APK：检查电视设置中是否开启了未知来源应用安装权限。

⚠️ **注意事项**

影视仓本身不含任何影视资源，所有内容通过配置接口获取。接口来源的合规性由源站负责，建议通过合法渠道获取内容。

百度网盘 Cookie 包含你的账号凭证，不要分享给他人。影视仓是本地应用，Cookie 只存储在你的设备上。

接口检测站 [https://tvbox.clbug.com/](https://tvbox.clbug.com/) 是社区维护的工具，接口的可用性和时效性会变化，建议定期更新。

影视仓的 GitHub 项目地址是 CatVodTVOfficial/TVBoxOSC，注意辨别真假。
