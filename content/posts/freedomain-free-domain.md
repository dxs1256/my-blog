注册免费域名通常意味着各种套路——免费一年续费天价、有广告有水印、限制一堆。最近发现一个 GitHub 上 162K Star 的项目 FreeDomain，主打一个真正的免费。

项目地址：[https://github.com/DigitalPlatDev/FreeDomain](https://github.com/DigitalPlatDev/FreeDomain)

---

## 有什么特别的

FreeDomain 的核心卖点就是"没有套路"：

- 注册免费、续费永久免费，没有隐藏费用
- 没有水印、没有广告、没有使用限制
- 个人、组织、商业项目都能用
- 兼容 Cloudflare、FreeDNS 等主流 DNS 服务商
- 后缀选择还挺多的，不像其他免费域名只有奇奇怪怪的尾缀
![FreeDomain 项目截图](https://external-content.duckduckgo.com/iu/?u=https://pbs.twimg.com/media/HJKoNcjbIAAMUav.jpg?name=orig)

## 适合什么场景

- 个人博客：不想花钱买域名又想有一个正经的网址
- 开源项目演示：临时上线一个 demo，不需要花钱买域名
- 临时测试环境：项目还没成型前先用免费域名顶着
162K Star 说明社区认可度很高，这种闷头做公益的项目确实稀缺。

## 怎么用

去 GitHub 仓库的 README 里找注册入口，按照指引操作即可。因为兼容 Cloudflare，拿到域名后可以直接托管到 Cloudflare 上统一管理。
