买 VPS 就像开盲盒。商家宣传页上写的是"高性能""大带宽"，到手一测才发现 CPU 跑分拉胯、网络绕了大半个地球、IP 还要被各种风控拦截。更烦的是，你永远不知道这台机器在别人手里到底是啥表现。

后来发现了一个叫 **NodeQuality** 的平台，一个问题全解决了——不光能自己测，还能搜到别人测过的结果，买 VPS 之前先看一眼，心里有底。

项目地址：https://nodequality.com/

## 🎯 这是啥

NodeQuality 是一个开源的服务器基准测试平台，核心就一件事：**用一行命令跑完整测试，结果全网公开**。

测完之后，你的测试结果会出现在平台的搜索结果里，其他人搜同一个主机商就能看到。相当于每个人都在帮大家做实测，越测数据越丰富。

## ⚡ 怎么用

简单到离谱。SSH 进你的服务器，跑这一行就行：

```bash
bash <(curl -sL https://run.NodeQuality.com)
```

不需要装任何依赖，不需要配置，脚本跑完自动出结果。

## 📊 测什么

NodeQuality 的测试覆盖三块核心数据：

| 测试维度 | 具体内容 |
|----------|----------|
| **硬件信息** | CPU 型号、核心数、内存大小、硬盘读写速度 |
| **IP 质量** | IP 类型（原生/广播/机房）、风控评分、是否被墙 |
| **网络质量** | 下载/上传速度、延迟、路由路径、国际带宽 |

最实用的其实是 **IP 质量检测**。很多便宜 VPS 的 IP 被各种服务商标记为"机房 IP"，买来发现注册不了 OpenAI、Netflix 看不了，甚至发邮件都被拒收。NodeQuality 直接告诉你这个 IP 干不干净。

## 🔍 搜别人测过的

这是 NodeQuality 最有价值的功能——**搜索结果公开可查**。

在首页搜索框输入主机商名称（比如 "Oracle""RackNerd""Vultr"），就能看到所有该商家用户在近期跑过的测试结果。每个结果包含测试 ID、ASN 归属、测试时间，点击还能看详细数据。

买 VPS 之前先去搜一下，看看真实用户的实测数据，比看商家宣传页靠谱十倍。

## 🔒 安全和隐私

脚本在临时系统中运行，测试完成后所有痕迹自动清除，对原系统环境没有影响。平台会收集 IP 地址和测试时间用于防滥用，但不会向第三方出售个人数据。

支持几乎所有的 Linux 发行版，无需担心兼容性问题。
